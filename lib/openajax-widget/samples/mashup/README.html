About the Sample Mashup Application

The Sample Mashup Application demonstrates the use of OpenAjax Metadata 1.0 and OpenAjax Hub 2.0 within a mashup assembly application.

The source code for this application can be found in the OpenAjax Alliance's open source project at SourceForge.

The mashup application was created in 2008 as the centerpiece for the alliance's 2008 InteropFest interoperability event.
