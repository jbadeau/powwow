{Object} create(args)
Creates an instance of a widget and adds it to the page, loading resources as needed.
Parameters:
{Object} args
Parameters used to create a widget. This object may contain the following properties:
{Object | String} args.spec
A widget spec object (as returned by OpenAjax.widget.Loader#loadMetadata) or a URL pointing to the widget specification.
{HTMLElement | String} args.target
HTML DOM element or ID of element in which this widget is rendered.
{String} args.id Optional
A string identifier for this widget. If null, one is generated.
{String} args.mode Optional
Initial mode of rendered widget. If not specified, defaults to "view".
{String} args.locale Optional
The locale to be used when doing localization substitutions. If not specified, defaults to the browser locale.
{Object} args.properties Optional
Initial widget properties. This is an object of name-value pairs, where the name matches a property name as defined in the widget metadata.
{Object} args.availableDimensions Optional
The maximum size to which a widget can be sized inside of this mashup framework. This object has "width" and "height" properties. If either is omitted, then that dimension is considered unconstrained.
{Function} args.adjustDimensions Optional
A callback which gets invoked whenever a widget requests a resize. This function takes two parameters: a widget ID and a dimensions object, with "width" and "height" properties. If the callback returns null, the resize request is ignored. Otherwise, the callback should return a dimensions object with dimensions less than or equal to those passed in, in order to size the widget to those dimensions.
{Boolean} args.sandbox Optional
If true, will load the widget as sandboxed (in an iframe), regardless of what the "sandbox" attribute is set to in the widget spec. If false or unspecified, then we honor the "sandbox" attribute as specified in the widget spec.
{Function} args.onComplete
Callback which is invoked if loading the URL was successful, onSuccess(metadata).
{Function} args.onError
Callback which is invoked if an error occurs, onError(error).
Returns:
{Object} Widget instance object.
